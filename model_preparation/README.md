# Model Preparation Module

This module converts the processed IEEE-CIS dataset into model-ready train and validation files.

## Current Scope

- loads a processed IEEE-CIS CSV generated by the data preprocessing layer
- removes the target and ID columns from feature inputs
- performs a stratified train-validation split
- groups rare categorical values into a shared `rare` bucket
- imputes numeric and categorical missing values
- encodes categorical columns with `OrdinalEncoder`
- saves train and validation datasets for model training
- saves the fitted preprocessing pipeline as a reusable artifact

## Folder Layout

- `scripts/`: model-preparation scripts
- `artifacts/`: saved preprocessing pipelines
- `outputs/datasets/`: train and validation datasets
- `outputs/reports/`: preparation reports and summaries

## Run

Using the IEEE development sample:

```powershell
python model_preparation/scripts/prepare_ieee_model_data.py --input-path data_preprocessing/outputs/ieee_cis/processed/ieee_cis_dev_sample_processed.csv --output-prefix ieee_dev_model_ready
```

Using the full IEEE dataset:

```powershell
python model_preparation/scripts/prepare_ieee_model_data.py --input-path data_preprocessing/outputs/ieee_cis/processed/ieee_cis_full_train_processed.csv --output-prefix ieee_full_model_ready
```

## Outputs

- train dataset CSV
- validation dataset CSV
- fitted preprocessing artifact in `joblib` format
- JSON report with split sizes, feature counts, and fraud ratios
